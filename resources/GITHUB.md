# GitHub for Groups:
*Be sure to create a branch for every new feature you implement.*

## General Workflow:
Adding a new feature:

* **Create a new branch:**
  ```bash
  git branch <branch-name>
* **Implement your new feature making commits to this new branch**
    ```bash
    git commit -m "[tag] <description-of-commit>"
* **Create a ‘pull request’ inside the browser or using a VS Code extension**
   
   * *we will then review the code and resolve any conflicts before pushing to main*
